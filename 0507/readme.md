ZFS
안정성과 성능, 그리고 데이터 보호 기능들이 강화된 파일 스토리지를 목적으로 개발

1.스토리지 풀
볼륨 확장, 디스크 추가시 스토리지에 결합하며 결합 직후 직시 용량이 반영 되어 사용가능
